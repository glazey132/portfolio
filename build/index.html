About

I'm a software engineer at Handshake - working on the Education team (a fully distributed/remote team!) - an up and coming startup focused on democratizing employment opportunities for students. I also sit on the DEI (Diversity, Equity and Inclusion) Task Force put together at Handshake with the goal of making Handshake a more diverse and inclusive workplace.

Originally from Minneapolis, Minnesota. I graduated from Stanford University with a degree in English - Creative Writing. I am a largely self-taught software engineer who blends a creative prowess with an analytical approach to solving problems.

I'm passionate about helping others and I have a love for making difficult things easy. I enjoy using code to help others make their ideas a reality. To do this, I build intelligent software with a focus on meaningful data, responsive designs, intuitive / aesthetically pleasing UIs, speed and useability.

Instagram Web Scraper

An Instagram web scraper built with Node.js / Express.js, Puppeteer.js and Kue.js. I built this project to gain a better understanding of web workers and queues. I used Puppeteer to provide headless browser navigation to crawl Instagram pages and scrape user information. Then, I set up a worker queue using KueJS since Kue provides an out-of-the-box job queue that allows us to continuously process scraped information upon it's availability.

React Calculator

This is a simple web-app that displays a working calculator with basic functionality. Working on this calculator was a fun and challenging project that helped me increase my knowledge of web sockets and real time data transfer. I built the UI with React, and created a Node.js Express server connected to a MongoDB, via mongoose, that saves all calculations and serves the 10 most recent calculations via a websocket (Socket.io) connection. The app and server are both hosted on Heroku but deployed as separate apps. The v1 of this project focused less on aesthetic, and more on gaining a perfect understanding of the connecting mechanics. But I have since updated the calculator to be a bit more visually appealing.
