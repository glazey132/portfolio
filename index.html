About

I'm a software engineer at Survata - an Ad Measurement and Market Research startup based in San Francisco.

Originally from Minneapolis, Minnesota. I graduated from Stanford University with a degree in English - Creative Writing. I am a largely self-taught software developer, who blends a creative prowess with an analytical approach to solving problems.

I'm passionate about helping others and I have a love for making difficult things easy. I enjoy using code to help others make their ideas a reality. To do this, I build intelligent software with a focus on meaningful data, responsive designs, intuitive / aesthetically pleasing UIs, speed and useability.

Trouvaille.io

Trouvaille is an unfinished medium to long distance social ride-share mobile application for use in the USA. I was one of 3 engineers who worked on this idea in a deadline driven app development workshop. While the project went unfinished, we finished a majority of the UI design and functionality. We built the frontend with React-Native, and AWS S3. While the backend server was made with Express, MongoDB, and Stripe. The server is hosted on Heroku.

Instagram Web Scraper

An Instagram web scraper built with Node.js / Express.js, Puppeteer.js and Kue.js. I built this project to gain a better understanding of web workers and queues. I used various packages to help with this: Puppeteer provides headless browser navigation to crawl Instagram pages and scrape user information. And, KueJS provides a helpful job queue that allows us to continuously process scraped information upon it's availability.

React Calculator

I built this calculator in React in a few hours as a speed test. I built the UI with React, and created an Express server with a MongoDB that saves all calculations and displays the 10 most recent calculations on the UI. Hosted on Heroku. This project focused less on aesthetic, and more on gaining a perfect understanding of the connecting mechanics.
